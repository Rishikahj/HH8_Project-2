# CTF Challenge Platform

This project is a web-based Capture The Flag (CTF) Challenge Platform designed to provide hands-on practice for basic cybersecurity concepts in a safe and controlled environment.

## Technologies Used
- PHP
- MySQL
- HTML, CSS
- XAMPP
- GitHub

## Development Environment
- Operating System: Windows
- Local Server: XAMPP (Apache, PHP, MySQL)

## Project Status
Initial environment setup completed. PHP is successfully running on the local server.

## Features
- Dynamic CTF challenges loaded from database
- Flag submission and validation
- Beginner-friendly challenge design
- Secure and educational environment

## How to Run the Project
1. Install XAMPP
2. Start Apache and MySQL
3. Place the project inside `htdocs`
4. Import the database
5. Open `challenges.php` in browser
